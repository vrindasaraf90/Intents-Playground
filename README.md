
# Intents-Playground

Created a app wih 2 activity.

Features
-Types of Intent (Explicit & Implicit)
-Explicit (increment and decrement button)
-Implicit (Open a webpage , Dial a number, Share a text)
-Sending and receiving data b/w 2 activity
